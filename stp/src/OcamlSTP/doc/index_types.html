A typed STP expression: the parameters of ('type, 'kind) expr are phantom types to enforces correct API usage: 'type is the type of the STP expression which may be a boolean [`bool], a bitvector [`bv], or an array [`array];, 'kind indicates whether the STP expression is an expression [`expr], a variable [`var], or a constant [`const].
